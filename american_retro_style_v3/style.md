# 美式复古漫画风格 Logo 设计要求 (V2)

## 生图方式
- 生图方式 text_to_image

## 提示词要求
### 固定部分
- American retro comic style, pop art, above the shoulders
### 其他要求
- 提示词内容要与官方素材**高度相关**,包括但不限于物品\职业特征

## 生图要求
- 1024x1024
- PNG 格式
- 肩部以上，头部占主体位置
- 合适的放射状光柱背景

## 主体角色要求
- 鸭子阵营：必须是动漫风格鸭子，表情邪恶/狡诈/罪恶方向
- 鹅阵营：必须是动漫风格鹅，表情可爱/善良/正义/阳光方向
- 中立阵营：根据实际物种漫画风格处理
- 可戴卡通化配饰体现职业

## 文件命名
```
{阵营}_{角色名}.png
```

## 合规红线
1. 禁止血腥/恐怖
2. 禁止真实危险物品
3. 绝对禁止任何边框
4. 必须保持卡通风格
5. 禁止暴力暗示
6. 提示词中避开出现职业类提示词,避免AI产生幻觉,通过职业特征将动物拟人化描述
